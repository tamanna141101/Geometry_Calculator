Frequently Asked Questions

It is a long established fact that a reader will be distracted by the readable content of a page when looking at its layout

DOM stands for Document Object Model, which is a programming interface for web documents. Using Dom Javascript can change all the HTML elements , attributes, CSS style in the page.

There are five methods for Select HTML elements using DOM--
1.getElementsByTagName()
2.getElementsByClassName()
3.getElementById()
4.querySelector()
5.querySelectorAll()

When an event happens on an element, like a click or a hover, it will also trigger the same event on all of its parent elements up to the top of the document. This is called event bubble.

A callback function is a function that is executed after another function is executed. And it is called when a certain event occurs. Using a callback function allows a program to continue executing while waiting for an external operation to complete, and then perform a specific action when that operation is finished.
